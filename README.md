# API 초기 세팅
pip install fastapi

pip install uvicorn

pip install pymysql

pip install python-dotenv

.env 파일생성후 db 정보 입력 (host, user, password, db)


# API 실행방법
1. API폴더로 이동 (cd API)
2. 명령어 입력 : uvicorn main:app --reload
